    Git&Hexo常见错误及解决方法

    配置 > 博客 git hexo
    发布时间 : 2021-05-18 16:03
    字数:386

      1. 10054
    2. git

    3. hexo

    errno

    10054

    • 报错位置 ：hexo init、git push

    • 提示信息：

      OpenSSL SSL_read: Connection was reset, errno 10054

    • 解决方法：

      • 网络问题，多次尝试；

      • 解决 ssl 验证，步骤如下：

        1. 打开 “Git Bash Here”；

        2. 输入如下命令：

          git config --global https.sslVerify "false"

        3. 重新进行hexo init操作


    git

    Please make sure you have the correct access rights and the repository exists

    • 报错位置：hexo d、git push

    • 提示信息：

      Please make sure you have the correct access rights and the repository exists

    • 解决方法：重配 SSH 公匙

    • 注意：SSH 是添加到 github 个人下的设置里的，不是 repository 下的设置


    hexo

    hexo s

    • 报错位置：hexo s

    • 提示信息：

      Failed to load resource: the server responded with a status of 404(Not Found)

(on the page)Cannot GET /

    • 解决方法：hexo 下 _posts 文件夹里如果没有 .md 文件（a post）就会出现这个问题，在文件夹中添加一个 post 就可以解决

    404

    • 报错位置：已经部署了的博客文章上
    • 提示信息：404
    • 具体情况：将.md文件名上的小写改成了大写，如hexo改成了Hexo
    • 解决方法：推断github上文件大小写不敏感，所以要采用其它可识别的修改方式进行文件名的修改，如增删字符、打空格等，再还原成改了大小写之后的文件名

    很久没有倒腾博客了，这次在另一台电脑上部署就出了这么多问题，后悔第一次搭建博客的时候没有把碰到的问题记录下来，花了好多功夫才成功部署，补上补上！
